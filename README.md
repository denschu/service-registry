Service Registry for integration platforms
================================================

[![Build Status](https://travis-ci.org/denschu/service-registry.png?branch=master)](https://travis-ci.org/denschu/service-registry)

* Caching
* GUI
* Metadata
* Persistence


# Installation

	cd service-registry
	mvn clean install

# Run

	cd service-registry
	mvn spring-boot:run
